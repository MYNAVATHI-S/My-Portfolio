Lamp Light Sphere Design

Design Methodology

Steps in Fusion 360

  1. Create a 50 mm sphere as reference
  2. Sketch and extrude ribs (50 mm height, 2 mm thick)
  3. Use circular pattern tool
  4. Create rings and cut slots
  5. Use join/cut for assembly
  6. Add fillets or chamfers

Design the Horizontal Rings

  1. Create a new sketch on the XZ plane.
  2. Draw a circle:
    • Outer diameter = 50 mm
    • Inner hole = 20 mm
    • Thickness = 2 mm
  3. Cut slots:
    • Slot width = 2 mm
    • Slots = number of ribs
  4. Extrude to 2 mm
  5. Create 2–3 rings (top, middle, bottom)

Dimensions and Number of Shapes

  • Sphere diameter: 50 mm
  • Ribs:
    • Number of ribs: About 8
    • Rib thickness: ~2 mm
    • Rib height: 50 mm
    • Rib spacing: 360° ÷ number of ribs

Basic Structure

  • Sphere: Defines the overall shape.
  • Ribs: Created using splines or arc-based sketches.
  • Rings: Created using circles and cutouts for rib slots.

LaserCAD

  • Right-click sketch → Save as DXF
  • Import DXF into LaserCAD for cutting

Assembly and Final Touch

  • Ensure ribs fit in slots
  • Adjust spacing/dimensions
  • Use Combine to merge parts
  • Apply Fillets for aesthetics
